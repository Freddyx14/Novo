┌─────────────────────────────────────────────────────────────────────────┐
│ 🎯 FLUJO DE USUARIO                                                      │
└─────────────────────────────────────────────────────────────────────────┘

1. REGISTRO
   └─→ Usuario visita /register
       └─→ Completa formulario (nombre, email, contraseña)
           └─→ Cuenta creada en Supabase
               └─→ Redirigido a /login con mensaje de éxito

2. INICIO DE SESIÓN
   └─→ Usuario visita /login
       └─→ Ingresa email y contraseña
           └─→ Sesión JWT creada
               └─→ Redirigido a /profile

3. USO DE LA APLICACIÓN
   └─→ Usuario en /profile (autenticado)
       └─→ Sube CV y audio
           └─→ Ve análisis en /results
               └─→ Explora oportunidades en /dashboard

4. CIERRE DE SESIÓN
   └─→ Usuario hace clic en "Cerrar Sesión"
       └─→ Sesión terminada
           └─→ Redirigido a /login

┌─────────────────────────────────────────────────────────────────────────┐
│ ⚙️ CONFIGURACIÓN REQUERIDA                                              │
└─────────────────────────────────────────────────────────────────────────┘

1. CREAR PROYECTO EN SUPABASE
   → https://supabase.com
   → Crear nuevo proyecto
   → Copiar URL y anon key

2. CONFIGURAR .env
   → SUPABASE_URL=tu_url_aquí
   → SUPABASE_KEY=tu_key_aquí
   → SECRET_KEY=clave_secreta_flask

3. EJECUTAR SQL EN SUPABASE
   → Abrir SQL Editor
   → Copiar/pegar supabase_auth_setup.sql
   → Ejecutar script

4. HABILITAR EMAIL AUTH
   → Authentication → Providers → Email
   → Habilitar Email provider

5. EJECUTAR APLICACIÓN
   → python3 test_setup.py  (verificar config)
   → python3 app.py         (iniciar app)
